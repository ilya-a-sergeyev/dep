Begin инструкция-маркер начала функционального блока

End инструкция-маркер окончания функционального блока

Push инструкция чтения содержимого клетки с указанного адреса и помещения его в стек
	операнд 1 - номер регистра внутренней памяти содержащего вектор-смещение относительно начала
	логического блока (инструкции Begin)
	операнд 2 - уровень вложенности блока содержащего Begin - 0 - текущий блок, 1 - вышестоящий блок и т.д.
	*операнд 1 инкрементируется в соотвествии с направлением из прочитанной клетки

Pop инструкция записи содержимого клетки по указанному адресу с вершины стека
	операнд 1 - номер регистра внутренней памяти содержащего вектор-смещение относительно начала
	логического блока (инструкции Begin)
	операнд 2 - уровень вложенности блока содержащего Begin - 0 - текущий блок, 1 - вышестоящий блок и т.д.
	*операнд 1 инкрементируется в соотвествии с направлением из записанной клетки
	*если уровень вложенности блока меньше 0 то происходит извлечение элемента из стека без записи и инкрементации операнда 1

Start инструкция запуска нового животного с указанного адреса
	операнд 1 - номер регистра внутренней памяти содержащего вектор-смещение относительно начала
	логического блока (инструкции Begin)

Len инструкция получения в регистр внутренней памяти количества инструкций в текущем существе
	операнд 1 - номер регистра внутренней памяти

GetE инструкция получения значения энергии указанной клетки в регистр внутренней памяти из верхнего значения стека (см. Push)
	операнд 1 - номер регистра внутренней памяти
	* для того чтобы узнать значение энергии клетки требуется последовательно выполнить инструкции Push и GetE

SetE инструкция записи значения энергии верхнему элементу стека (см. Push и Pop) в регистр внутренней памяти
	операнд 1 - номер регистра внутренней памяти
	* объем энергии существа уменьшается на записываемое число

CmpTop инструкция сравнения значения энергии указанной клетки со значением в регистре внутренней памяти
	операнд 1 - номер регистра внутренней памяти содержащего вектор-смещение относительно начала
	логического блока (инструкции Begin)
	операнд 2 - номер регистра внутренней памяти
	* выставляет флаги
